# 常用浏览器常用快捷键快捷键

常用浏览器: **Chrome** 和 **Foxfire**

## Chrome

| 命令       | Windows    | OS X     | Linux    |
| :-------- | :--------- | :-------- | :-------- |
| 放大字体 | `Ctrl` + `+` | `Cmd` + `+` | `Ctrl` + `+` |
| 缩小字体 | `Ctrl` + `-` | `Cmd` + `-` | `Ctrl` + `-` |
| 充值字体缩放 | `Ctrl` + `0` | `Cmd` + `0` | `Ctrl` + `0` |
| 页面后退 | `Alt` + `<-` |  | `Alt` + `<-` |
| 页面向前 | `Alt` + `->` |  | `Alt` + `->` |
| 新建Tab页 | `Ctrl` + `T` |  | `Ctrl` + `T` |
| 关闭当前页 | `Ctrl` + `W` |  | `Ctrl` + `W` |
| 关闭所有页 | `Ctrl` + `Shift` + `W` |  | `Ctrl` + `Shift` + `W` |
| 恢复上一次关闭页 | `Ctrl` + `Shift` + `T` |  | `Ctrl` + `Shift` + `w` |
| 新建窗口 | `Ctrl` + `N` |  | `Ctrl` + `N` |
| 新建私密窗口 | `Ctrl` + `Shift` + `N` |  | `Ctrl` + `Shift` + `N` |
